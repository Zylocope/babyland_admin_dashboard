# Appleland Admin Web

Admin panel for **Appleland**, a baby store & indoor playground in Naypyidaw, Myanmar.
Manages online orders, playground ticket bookings, products, customers, staff, and reports.

## Includes

- **Dashboard** — daily sales, revenue, pending/completed orders, low-stock alerts
- **Products, Orders, Playground Tickets, Customers, Reports, Staff** modules
- **Role-based access** — Manager (full) / Staff (view + process)
- **Light / Dark mode** and **English / Myanmar (i18n)** language toggle
- Warm-orange design system

## Tech

React + Vite · Tailwind CSS v4 · React Router · Recharts · Tabler Icons · i18next · date-fns

## Run

```bash
npm install
npm run dev      # 5173
npm run build    
npm run preview  
```
